Aula 3: Ajax com XMLHttpRequest

O Ajax é uma tecnologia usada para requisições assíncronas, ou seja, não existe sincronismo ente
as requisições, sendo assim, podemos enviar diversas requisiçõesem paralelo, onde cada resposta retorna
quando estiver pronta.
    1-no método selectAll() que antes buscava os dados salvos no localStorage vai buscar no servidor-restfull
    2-criar variavel let ajax = new XMLHttpRequest();
    3-ajax executa o métod open() que recebe um método e uma rota: ajax.open('GET', '/users')
    4-depois criamos um evento chamado de ajax.onload = event => {}
    5-por ultimo executamos o método ajax.send() que vai enviar a resposta do servidor restfull para o client
    de forma assíncrona
